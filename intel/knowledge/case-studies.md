# Production Case Studies

## Cathryn Lavery — E-Commerce ($55M)

- **Revenue**: $55M generated, 5M individual products sold
- **Setup**: Bought business back, using Paperclip to make it efficient with AI
- **Team**: Human + agent hybrid. Real human (Catarina) works alongside agent (Rory, named after Rory Gilmore)
- **Chief of Staff**: "Knox" — morning + evening health checks, manages blockers, autonomy log
- **Key insight**: "Fired half my agents" — fewer, better-structured agents beat many poorly-structured ones
- **Pain point**: "I don't like managing a human team very much, and I really don't want to manage an AI team either"
- **Human friction**: "My real human team doesn't want markdown files" → agent puts docs in Google Docs/Proof
- **SEO workflow**: Exported Ahrefs CSV → fed to Paperclip → auto-created issues → engineer fixes technical SEO, writer updates articles

## Francisco — Dentist/Oral Surgeon
